## Linux Enumeration Tool

This is a basic bash script to find enumerate the Linux Operating System.


## Current Features
- basic system information fetching like (architecture, cpu, memory, kernel version .etc.)
- user infromation (current users, active users, user groups .etc.)


## SETUP AND EXECUTION

#### 1. Setting up
- clone the repository.

```bash
git clone https://github.com/ghostware0x00/enumix.git
```

#### 2. Giving Permissions
- go to the cloned folder and give executable permissions to the `enumix.sh` script. 

```bash
chmod u+x enumix.sh
```

#### 3. Execution
- after giving executable permissions execute the `enumix.sh` script.

```bash
./enumix.sh
```
